* 변수 *
- 숫자 타입은 int
- 문자 타입은 String ( 참조형 변수 )


* 자료형 변수와 참조형 변수 (주요 메모리, 보조 메모리) *
- String 변수는 용량이 크기 때문에 주요 메모리에 담으면 오류가 날 수 있음
- String에 있는 위치를 참조해서 데이터를 불러온 후 보여주기 때문에 참조 변수라고 함
